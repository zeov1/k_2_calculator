Hello! It's a query-based calculator.

In order to calculate 2 float numbers, follow these steps:

  1. Type ? after "calc/"
  2. Specify the value of the parameter a like this: .../?a=10
  3. Type &. You have to do it between each pair of parameters
  4. Specify the value of the parameter b (similarly)

  5. Specify the operation (op) between a and b. The application supports

    • plus
    • minus
    • mult
    • div

In result, you create a GET-request like this:

https://localhost/calc/?a=12&b=5&op=div {% endblock content %}
